# مواصفات مؤشرات الأداء ولوحة التقارير

## التعاريف الأساسية
- **MRR (B2B):** مجموع الرسوم الشهرية للعقود الفعالة.
- **Repeat Rate:** الطلبات المتكررة / إجمالي الطلبات.
- **Cancellation Rate:** الطلبات الملغاة / إجمالي الطلبات.
- **SLA Compliance:** طلبات ضمن حدود الاستجابة/الإنجاز / إجمالي الطلبات المعنية.
- **Pipeline Coverage:** (قيمة الفرص خلال 90 يوم) / (الهدف الشهري × 3).

## أحداث تتبع مقترحة
- `lead_created`, `outreach_sent`, `meeting_held`, `proposal_sent`, `contract_signed`,
  `job_completed`, `job_canceled`, `invoice_sent`, `payment_received`.

## تقارير
- أسبوعي: صفقات، مراحل، احتمالات، عوائق.
- شهري: الإيراد، الامتثال للـSLA، الإلغاءات، التكرار.


# دليل منع التسرب وإلغاء الطلبات (Anti-Churn)

1) **داخل المنتج**
- تذكير مذهّب بعد إنشاء الطلب يوضح مزايا إتمامه داخل التطبيق (الضمان/الفوترة/الدعم).
- رسائل دفع ذكية وتأكيد الموعد + نافذة إعادة جدولة.
- تحذير قبل الإلغاء + تقديم حل بديل (خصم صغير/أولوية موعد/فني بديل).

2) **اتفاقيات المزوّدين**
- ربط مستحقات المزوّد بإتمام الطلب داخل التطبيق.
- حوافز للالتزام بالـSLA والرضا.

3) **عمليات التشغيل**
- متابعة يومية لحالات P1/P2.
- اتصال بشري للحالات المعرّضة للإلغاء خلال 2 ساعة.

4) **تحليلات**
- لوحة أسباب الإلغاء، نسبة التكرار، الرضا، زمن الاستجابة/الإنجاز.
# هياكل SLA وNDA (عربي/English) — مسودة أولية

## SLA — مستويات الخدمة (AR)
- **الاستجابة:** P1 خلال 2 ساعة / P2 خلال 4 ساعات / P3 خلال 8 ساعات.
- **الإنجاز المستهدف:** P1 خلال 12 ساعة / P2 خلال 24 ساعة / P3 خلال 72 ساعة.
- **توفر الخدمة/الدعم:** 6 أيام/الأسبوع (أو 24/7 للباقة المتقدمة).
- **التقارير:** شهريًا يشمل الامتثال للـSLA، الأعمال المنجزة، الإلغاءات، الشكاوى.
- **جزء الحوافز/الجزاءات:** اعتمادات ائتمانية عند الإخلال الجوهري / حوافز عند تجاوز المستهدف.

## NDA — أبرز البنود (AR)
- التعريفات، الغرض، السرية، السماحيات، الاستثناءات (المعلومة العامة/بملكية الطرف)، المدة (2–3 سنوات)، القانون واجب التطبيق، الاختصاص.

---

## SLA — Service Levels (EN)
- **Response:** P1 within 2h / P2 within 4h / P3 within 8h.
- **Resolution Target:** P1 within 12h / P2 within 24h / P3 within 72h.
- **Support Availability:** 6 days/week (or 24/7 for premium).
- **Reporting:** Monthly report covering SLA adherence, completed jobs, cancellations, complaints.
- **Incentives/Penalties:** Credits for material breach; bonuses for exceeding targets.

## NDA — Key Clauses (EN)
- Definitions, Purpose, Confidentiality, Permitted Use, Exceptions, Term (2–3y), Governing Law, Jurisdiction.

> ملاحظة قانونية: هذه مسودات أولية للاسترشاد ويجب اعتمادها قانونيًا قبل الإرسال للعملاء.

- المشكلة: تباين جودة الخدمة، إلغاءات، غياب مؤشرات قابلة للقياس.
